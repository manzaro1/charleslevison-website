# Charles Levison Art Gallery Website

E-commerce website for Charles Levison - Visual Artist from Malawi.

## Features
- Complete e-commerce website with 4 pages (Home, Gallery, About, Contact)
- Professional styling with navy + gold theme
- Email marketing automation templates
- Art buyer list and newsletter system
- Mobile responsive design

## Setup
1. This site can be deployed using GitHub Pages
2. Replace image placeholders with actual artwork
3. Configure email automation for art collectors

Built with HTML, CSS, and vanilla JavaScript.
